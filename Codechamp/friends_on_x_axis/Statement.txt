Problem :

There are some boxes placed on x-cordinate axis. Position of each box is given.
You are given Q queries.
For each Query, print the number of boxes between L to R (including L,R).

Input :
The first line of input contains a single integer 'N' - Number of Boxes.
Second line contains N space separated integers denoting position of the box.
Third line contains Q - Number of queries.
Then input consist of Q lines, each containing two space separated integers L and R.

Output :
For each query, print the corresponding answer.

Constraints :
1<=N<=100000
1<=Q<=100000
1<=P<=1000
1<=L<=1000
L<=R<=1000

Sample Input :
5
1 3 4 6 2
3
1 3
2 6
5 5

Sample Output :
3
4
0
